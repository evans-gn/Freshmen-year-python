# Freshmen Year Python Projects

A collection of Python projects built during my freshman year.

Built by Evans | Freshman Year

## Projects
- Python Calculator - A calculator that performs addition, 
subtraction, multiplication and division
- Weight Converter - A program that converts weight between 
kilograms and pounds
- Mad Libs - A fun fill in the blank story game using user input
- Temperature Converter - A program that converts temperature 
between Celsius and Fahrenheit
- Compound Interest Calculator - A calculator that computes 
compound interest with input validation
